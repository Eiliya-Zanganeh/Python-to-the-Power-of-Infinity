## Exercise 1: Layer Sequence Validation Program

Write a program that takes a layer's angle, type, and weight as input and checks whether the input is valid. If all inputs are correct, print a success message. If not, print the items that were entered incorrectly so the user knows what they got wrong.

**Notes:**

- The angle must be one of `["90", "45", "0", "-45"]`. Otherwise, it's incorrect.
- The type must be one of `["Woven", "UD"]`. Otherwise, it's incorrect.
- The weight must be one of `["200", "300", "400"]`. Otherwise, it's incorrect.

## Exercise 2: Health Diagnostic Program

Write a program that takes body temperature, heart rate, and breathing rate as input and checks whether all conditions are balanced. If everything is correct, display a "correct" message. If not, print the items that need to be reviewed.

**Notes:**

- Body temperature must be between 37.5 and 38.5. Otherwise, this item needs review.
- Heart rate must be between 44 and 60 beats per minute. Otherwise, this item needs review.
- Breathing rate must be between 8 and 16 breaths per minute. Otherwise, this item needs review.

## Exercise 3: Lead Quality Evaluation Program

Write a program that takes three key marketing lead criteria as input and checks whether the lead is ready to be sent to the sales team. If all criteria are in optimal status, print "Lead is qualified — send to sales department." If not, display the items that need improvement.

**Notes:**

- Email open rate (%) must be a number between 20 and 100. Otherwise, this item needs review.
- CRM lead score must be a number between 70 and 100. Otherwise, this item needs review.
- Customer journey stage must be one of `["Awareness", "Consideration", "Decision"]`. Otherwise, this item needs review.

## Exercise 4: Houseplant Health Check Program

Write a program that takes three health indicators of a houseplant as input and checks whether the plant is in ideal condition. If all indicators are correct, print "The plant is perfectly healthy!" If not, display exactly which items have problems.

**Notes:**

- Soil moisture must be one of `["Dry", "Semi-moist", "Moist"]`. Otherwise, this item needs review.
- Leaf condition must be one of `["Green and lush", "Green", "Glossy"]`. Otherwise, this item needs review.
- Room temperature (°C) must be a number between 18 and 26. Otherwise, this item needs review.
