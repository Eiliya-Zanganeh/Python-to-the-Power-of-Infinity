# 🎬 Cinema Reservation System

## 📖 Introduction

In this project, you will build a simple **Cinema Reservation System** using Python.

The goal of this exercise is **not** to write the entire project from scratch. Instead, you will complete the missing parts marked with **TODO** comments throughout the code.

By the end of this project, you should be able to work with files, JSON data, functions, loops, conditions, lists, dictionaries, modules, and many built-in Python features.

# 📂 Project Features

Your program should support the following features:

* Show all movie schedules
* Add a new movie schedule
* Update an existing schedule
* Delete a schedule
* Reserve a seat
* Show available schedules
* Search for a movie
* Show unique movie titles

# 📝 Instructions

1. Read the entire source code before making changes.
2. Search for every `TODO` comment.
3. Replace every `...` with the correct Python code.
4. Do **not** change the program structure unless it is absolutely necessary.
5. Test your code after completing each TODO.
6. Make sure every menu option works correctly.
